W3schools

5/5

- This website provides well written explainations that are easy to understand. They also provide examples, and even have a "try it yourself" editor you can use.

CSS-Tricks

5/5

- As the name implies, this site provides a number of tricks you can use in CSS that you might not have thought of if you had just been using tutorials and textbooks. It's my go to source for how to use Flex Boxes.

Tutorialspoint

3/5

- The tutorials here are similar to W3schools, but go into more technical detail at the expense of readability.

Stack Overflow

4/5

- An excelent place to go if you need very specific answers to very specific questions.

Wikipedia

3/5

- This isn't so much for coding as it is for how the underlying systems work. If you want to know what something like FTP or HTTPS means, wikipedia will provide a thourough (if somewhat verbose) explaination.
